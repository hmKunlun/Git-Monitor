### 支持的Git命令
- git init：初始化Git仓库
- git clone：克隆远程仓库
- git branch：创建、列出或删除分支
- git checkout：切换分支或恢复工作区文件
- git add：添加文件到暂存区
- git commit：提交暂存的修改
- git merge：合并分支
- git push：推送到远程仓库
- git pull：从远程仓库拉取
- git cherry-pick：选择性应用提交

### 主要功能
- 实时监控Git操作
- 记录和汇总Git命令
- 分析提交内容和变更
- 生成仓库统计报告
